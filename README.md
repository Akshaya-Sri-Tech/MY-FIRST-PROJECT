**Tea Webpage** – Git Practice Project
This is a simple HTML webpage created primarily to practice Git and GitHub workflows, not as a full-scale web project.
📌** Purpose**
The main goal of this project is to gain hands-on experience with:

**Git commands**
Version control concepts
Working with GitHub repositories

🛠 **What I Practiced**
While working on this project, I used:

git init – to initialize a repository
git add – to stage files
git commit – to track changes
git status – to check changes
git log – to view commit history
git push – to upload code to GitHub
git pull – to sync changes

🌐 **About the Code**
The project includes a basic HTML file that:
Displays information about tea
Lists ingredients and preparation steps
Shows an image

🚀 **GitHub Usage**

Created a remote repository on GitHub
Connected local repo using git remote add origin
Pushed code using git push -u origin main

📖 Note
This is a beginner-level learning project focused on understanding Git and GitHub rather than building a production-ready webpage
